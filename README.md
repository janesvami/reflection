Задачи на Java Reflection:

1. Создайте класс private полями и public конструктором, который иницализирует их
2. Добавьте в классы приватные методы (

например

private String getPassword () {
return "verySecretPass";
}

)

и пару других на свое усмотрение

3. Создайте инстанс в приложении, инициализировав его.
4. Задания:

- получить список имен всех полей класса и их типы и вывести их
- вывести значения всех полей
- поменять значения некоторых полей на новые
- вывести имена и возвращаемые типы всех методов
- вызвать приватный метод класса через Reflection (смотри в сторону Reflection ivoke())
- создать новый инстанс класса, используя Reflection (не через new)

Доп.задание:

1. Создать новую аннотацию @Investigable
2. Навесить её над своим новым классам
3. Сделать так, что вся логика, описанная в пунктах выше применялась только к классам с аннотацией @Investigable, а к другим классам - нет

если на вход в ваш рефлекшн передаётся объект класса без этой аннотации, выводить "Class " + className + " is not investigatable" вместо всей инфы
